# Quick Start Guide - Restaurant Management System

## ⚡ Getting Started (30 seconds)

### Option 1: Browser-Based (No Installation Required)
1. Open the folder: `c:\Users\pc\OneDrive\Desktop\Restourant management system`
2. Double-click **`index.html`**
3. Done! Your restaurant management system is running 🎉

### Option 2: With Node.js Backend (Optional)
1. Install Node.js from https://nodejs.org
2. Open Command Prompt/PowerShell in the project folder
3. Run: `npm install`
4. Run: `npm start`
5. Open browser to: http://localhost:3000

## 🎯 Common Tasks

### Creating Your First Order
1. Click **"Tables"** in sidebar
2. Click any **green table**
3. Select menu items and quantity
4. Click **"Add to Order"**
5. Click **"Complete Order"** button
6. ✅ Order complete & billing generated

### Adding a Table
1. Go to **"Tables"**
2. Click **"+ Add Table"**
3. Fill in: Table Number, Capacity, Type
4. Click **"Add Table"**

### Adding Menu Items
1. Go to **"Menu"**
2. Click **"+ Add Item"**
3. Fill in: Name, Description, Price, Category
4. Click **"Add Item"**

### Checking Revenue
1. Go to **"Dashboard"** (default page)
2. See **"Today's Revenue"** card at top
3. Click **"Reports"** for detailed analytics

### Managing Inventory
1. Go to **"Inventory"**
2. Watch **color-coded stock bars:**
   - 🟢 Green = Healthy stock
   - 🟡 Yellow = Low stock warning
   - 🔴 Red = Critical, order now!

## ❓ Troubleshooting

### Page shows blank/error
- Refresh browser (F5)
- Clear cache (Ctrl+Shift+Delete)
- Try different browser

### Can't add tables/items
- Fill all required fields (marked with *)
- Try refreshing page
- Check browser console for errors

### Data disappeared
- Clear browser cache causes data loss
- Use incognito/private mode to avoid this
- Server version is safer (see Option 2 above)

### Want to reset all data
- Click browser settings
- Go to "Clear browsing data"
- Select "Cookies and cached files"
- Reload page
