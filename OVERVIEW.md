
## 🌍 Customization Options

### Colors
Edit in `styles.css`:
```css
--primary: #667eea;    /* Main brand color */
--secondary: #764ba2;  /* Secondary color */
--accent: #ffd700;     /* Highlight color */
```

### Restaurant Name
Edit in `index.html`:
```html
<h1 class="logo">🍽️ YourRestaurantName</h1>
```

### Menu Items
Edit in `app.js`:
```javascript
menuItems: [
    { id: 1, name: 'Your Item', price: 10.99, ... }
]
```
